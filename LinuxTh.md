### Горячие клавиши:

Ctrl + Alt + F3/F4/F5/F6 — переход в полностью консольный интерфейс;  
Ctrl + Alt + F2 — возврат к оконному режиму;  

### Регулярные выражения:

##### Позиция в строке (якоря)
^ — начало строки. Выражение ^Linux найдет строки, которые начинаются со слова Linux.  
$ — конец строки. Выражение done$ найдет строки, которые заканчиваются на done.  
^$ — пустая строка (символ начала сразу переходит в символ конца).  

##### Поиск символов
.(точка) — один любой символ (буква, цифра, пробел).  
\[abc] — любой один символ из указанных в скобках (или a, или b, или c).  
\[a-z] — любая строчная буква от a до z (диапазон).  
\[0-9] — любая одна цифра.  
\[^0-9] — любой символ, кроме цифры (знак ^ внутри квадратных скобок
означает отрицание).  

##### Количество повторений (квантификаторы)
\* — ноль или более повторений предыдущего символа.  
\+ — одно или более повторений (доступно в ERE).  
? — ноль или одно повторение (предыдущий символ либо есть, либо его нет).  
{n} — ровно n повторений символа.  

### Стандартные команды:

uptime — показывает как долго система работает;  
uname -a — информация о ОС;  
lscpu — данные о процессоре;  
ls -l \ -la \ -la -R / \— показывает имена файлов и директорий. Флаг -l дает более развернутую информацию содержимом текущей директории, а -la дополнительно показывает скрытые элементы. -R / просматривает все элеметы на всех дисках, к которым linux имеет доступ;  
echo $PATH — показывает пути к директориям из которых linux берет команды (при желании можно создавать новые директории и авторские команды);  
echo "*text*" — вывод текста в терминал (кавычки опцианальны);  
sudo — super user do, выполнение от имени администратора;  

### Работа с файлами и директориями, информация и навигация:

man — доступ к системным справочным страницам;  
info — чтение Info документации;  
whatis — однострочное описания чего-либо;  
whereis — найти файлы исполняемого кода, исходного кода и справочной страницы для команды;  
locate — быстрый поиск файлов по имени;  
find — поиск файлов в иерархии директорий;  
cd — смена директории (cd без указания пути переключит в домашнюю директорию (~));  
cd / — перейти в корневую директорию;  
cd .. — перейти на директорию выше (можно записать ../../.. и тд);  
pwd — путь к текущей директории;  
cat — вывод содержимого файла;  
more — открывает вьюер для просмотра содержимого файла;  
less — открывает вьюер для просмотра содержимого файла с возможностью поиска отдельных слов;  
sort — выводит содержимое файла в отсортированном по символам виде (флаг -n отсортирует по номерам);  
cut — используется для «вырезания» колонок или частей текста из каждой строки файла или потока данных.  
Флаги `-d` и `-f` работают в паре:
- **`-d` (delimiter / разделитель):** Указывает символ, который разделяет колонки в тексте (по умолчанию это знак табуляции).
- **`-f` (fields / поля):** Указывает номера колонок (полей), которые нужно извлечь;  
grep — используется для поиска строк, которые соответствуют заданному шаблону (тексту или регулярному выражению), в файлах или потоке данных;  
wc — показывает сколько строк/слов/символов в файле;  
touch — создать файл (обновить время элемента если применить команду на существующий файл или директорию);  
cp *filename* *dirpath* — копирование файла. В конце можно указать имя копии;  
cp filename*.* *dirpath* — копирование всех файлов с именем, в имени которого содержится filename;  
rm — удалить файл;  
mv — переместить файл или директорию или изменить имя;  
mkdir (*exsistdir*/*newdir*)(-p *newdir1*/*newdir2*) — создать директорию;  
rmdir — стереть пустую директорию;  
rm -R *dirname* — рекурсивно стирает всё содержимое указанной директории;  
cp -R *outdir* *indir2* — копирование всей директории из одной в другую;  
sudo rm -R / --no-preserve-root — даблэй волки хотят падвигаца;  
ln -s — создать линк на файл/директорию;  
ln — создать точную связанную с оригиналом копию;  
ln -s — создать линк на файл/директорию;  
ln *originame* *copyname*— создать точную связанную с оригиналом копию;  

### Архивирование:

tar cf *tarname*.tar *foldername* — заархивировать папку;  
tar xf *tarname*.tar — разархивировать архив;  
gzip/bzip2/xz *filename* — скомпрессировать файл;  
gunzip/bunzip2/unxz *filename* — раскомпрессировать файл;  
tar czf *archivename*.gz/cjf *archivename*.bz2/cJf *archivename*.xz — сжать папку;  
tar xf *archivename*.gz/cjf *archivename*.bz2/cJf *archivename*.xz — распаковать архив;  
tar tf *archivename* — посмотреть содержимое архива;  
zip -r *archivename*.zip *foldername* — запаковать папку в zip-архив;  
unzip *archivename*.zip — распаковать zip-архив;  

### Перенаправление ввода/вывода:

\> — создать новый файл с выводом;  
\>> — добавить в файл;  
2> — создать новый файл и записать плохие выводы (is a directory, permission denied, etc);  
2>> — добавить плохие выводы в файл;  

### Отслеживание метрик:

ping *address*  — считывание пинга сервера по указанному адресу (Ctrl + C — прервать запись, Ctrl + Z — остановить вывод пинга в терминал (запись продолжится в фоновом режиме) (команда fg — возобновить вывод));  

### Процессы:

top — отображает все действующие процессы и загруженность системы в реальном времени;  
free — показывает состояние памяти в байтах;  
free -h — показывает состояние памяти в Mb, Gb;  
ps — показать снимок запущенных из терминала процессов в реальном времени;  
ps aux — показывает **полный список всех запущенных в системе процессов** в реальном времени.
- **`a`** — показать процессы **всех пользователей** (а не только ваши собственные).
    
- **`u`** — вывести подробную информацию в **ориентированном на пользователя** формате (показывает имя владельца, загрузку CPU, памяти и т.д.).
    
- **`x`** — показать процессы, **не привязанные к терминалу** (фоновые службы, демоны и системные процессы, которые работают сами по себе);
